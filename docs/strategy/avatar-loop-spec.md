# Avatar Loop Specification v1 (RES-5)

## Role of Avatar

Avatar is not decoration. It is a behavioral anchor that visualizes the user's
future identity and links daily actions to visible progress.

## Input Layer

1. User media: photo or short selfie video.
2. Identity goals: selected aspiration tracks.
3. Daily evidence: action completion, self-reported state, streaks.

## Output Layer

1. Daily avatar message in the voice of "best self".
2. Micro visual changes tied to consistency milestones.
3. Weekly reflection snapshot: current self vs projected self.

## Reinforcement Loop

1. Forecast gives context.
2. Avatar message gives emotional pull.
3. Action card gives concrete step.
4. Completion updates avatar state.
5. Updated avatar increases next-day motivation.

## Risk Controls

1. Avoid manipulative certainty language.
2. Do not imply medical outcomes.
3. Show confidence and uncertainty labels for generated guidance.
