# User Flow v1: Birth Date -> Forecast -> Daily Loop (RES-4)

## Funnel

1. Entry
- User installs app and sees 20-second value narrative.

2. Onboarding
- Inputs birth date, time (optional), and location (optional).
- Chooses aspiration tracks: wealth, health, confidence, relationships.

3. First Reading
- App generates daily interpretation with confidence label and tone options.

4. Avatar Initialization
- User uploads photo or records short video.
- App creates a "best-self" avatar profile with selected goals.

5. Daily Action Card
- User receives one coaching action (2-10 min).
- User receives one mindfulness action (1-5 min).

6. Completion and Feedback
- User marks completion, logs mood/energy, and sees avatar progression cue.

7. Re-engagement
- Streak, reminder, and next-day narrative teaser.

## Activation Targets (v1)

1. Time-to-value: less than 2 minutes to first meaningful output.
2. Action completion day 1: greater than 35%.
3. Return intent day 1 survey: greater than 60%.
